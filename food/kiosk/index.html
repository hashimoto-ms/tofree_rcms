Features

ソリューションの特長

POINT

01.

直感的な操作感で
ストレスフリー

操作性に優れた縦型の大型カラー液晶ディスプレイで直感的な操作が可能です。

POINT

02.

柔軟な画面遷移での
アップセルによる客単価向上

注文画面の遷移の中で、サイドメニューのご提案や、よくセットで頼まれるオプションのオススメなどが可能です。
アップセルに繋がりにくい、という弱点を解消しています。

POINT

03.

モバイルオーダーとの連携も可能

モバイルからの注文も一元管理ができます。
店内からも店外からも注文が可能です。

POINT

04.

優れた現金処理機能で
締めの時間の削減

釣銭機で現金の入出金を自動計算します。
オープン前のつり銭準備や、閉店後の売上金回収に関わる業務時間を圧倒的に削減できます。
現金誤差を限りなくゼロにします。

POINT

05.

クラウド連携で簡単管理
アップデートによって常に最新機能が
使えます

クラウドシステム上に連携しているため、メニュー配信や売切変更、売上管理などの管理業務を遠隔からも実施可能。
店舗業務の負担を最小限にできます。

POINT

06.

24インチの大画面を
サイネージとして活用

待機画面やバナーなど空いているスペースに広告表示が可能です。
期間限定商品やおすすめ商品を効率よく告知できます。

POINT

07.

インバウンド対応も万全
多言語表示・音声ガイド

海外からのお客様にも、表示切替・音声ガイドでしっかり対応。
売上げアップのチャンスを逃しません。

POINT

08.

多彩な決済手段に対応

現金やクレジット、QRコード決済といった、あらゆる決済方法に対応しています。

FGK-100シリーズは色々なシステムと連携可能！

FGK-100シリーズは搭載アプリの汎用性が高く、外部の様々なシステムと連携することができます。

注文情報は「キッチンディスプレイ」「キッチンプリンタ」でいつでも確認

様々な経路からくるご注文は「キッチンディスプレイ」「キッチンプリンタ」で一元管理が可能です。
キッチン内で複数のシステムを同時管理する必要がなく、調理の順番やオペレーションの段取りをスムーズに決定できます。

O:derサイネージで呼び出し

様々な経路から来るご注文とその受け渡し方法は「キッチンディスプレイ」「キッチンプリンタ」で一元管理が可能です。
キッチンディスプレイからの操作で、商品受け渡しの際のお客さ呼出が可能です。
